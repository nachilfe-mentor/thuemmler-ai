Was ist Base64 und wann brauche ich es?

Was ist Base64?

Base64 ist ein Kodierungsverfahren, das beliebige Binärdaten in einen Text aus 64 ASCII-Zeichen (A–Z, a–z, 0–9, +, /) umwandelt. So können Binärdaten sicher in Textkontexten übertragen werden.

Bilder in HTML einbetten

Mit Base64 kannst du kleine Bilder direkt ins HTML oder CSS einbetten: src="data:image/png;base64,..." — kein separater HTTP-Request nötig.

E-Mail-Anhänge

E-Mail-Protokolle (MIME) verwenden Base64, um Binärdateien wie PDFs oder Bilder als Anhang zu übertragen. Dein E-Mail-Client kodiert und dekodiert das automatisch.

APIs und Authentifizierung

HTTP Basic Authentication überträgt Zugangsdaten als Base64-String im Header. Viele APIs nutzen Base64 für Tokens oder Binärdaten in JSON-Feldern. Achtung: Base64 ist keine Verschlüsselung!

Sicherheitshinweis: Base64 ist keine Verschlüsselung — kodierte Daten können von jedem einfach dekodiert werden. Für Passwörter und sensible Daten niemals Base64 allein verwenden. Verwende echte Verschlüsselung (AES, RSA) oder Hashing (bcrypt, Argon2).
